我的定位

发表于 2017-03-19   |   分类于 随笔

有时候我感觉我自己是一个极端的人，总想着：要么名满天下，要么死的其所。因此害怕唯唯诺诺，害怕止步不前….我也在寻找，我相信能在此生遇到最幸运的事：发现自己生来的使命，即便现在庸庸碌碌，即便在如此强大的社会潮流中，看不到自己的未来。

上了两年大学，我时常会感觉到害怕，感觉到焦虑，甚至感觉到恐惧。经常一个人熬夜，听着舍友的呼噜声，脑海中一幕幕想着未来的那个自己。我的专业是计算机，我也不知道怎么的就阴差阳错的来了，但说实话真的不喜欢。曾想过放弃，但又怕耽误自己，委屈自己，又怕接下来的选择会更不尽如人意。也就一直这样浑浑噩噩的学习着。自我感觉我是一个极端的人，不忍放弃，不会妥协，不止前行……如果有人问我“你为什么要上大学？”我会回答：“寻找自己的人生定位。”在两年的时间里我认识了很多有志同道合的同伴，或许可以叫同志吧，在他们的带领下，我一步步的走了下来，有这么一群家伙在身边，真好！

很久没有跟自己对话了，尝试回答一些过去要回避的问题。
1.介绍一下你自己，你是个什么样的人？
2.你有什么理想吗？这个理想是怎么形成的？
3.你理想的伴侣关系是什么样的？你自己在这个伴侣关系中扮演什么样的角色？要承担什么样的责任？
4.你理想的事业是什么，你正在做的工作符合你的事业理想吗？这份工作对你的意义是什么？
5.你对钱怎么看，你认为赚到多少钱是足够的？如果你明天一早醒来，已经有足够的钱，你将会如何继续安排自己的生活？
6.你的择友标准是什么？什么样的人你会愿意交往，什么样的人你会拒绝他交往？
7.你对死亡怎么看？如果你要立遗嘱，这份遗嘱会怎么写？

我相信，在我的身边也有那么一群人，在忙忙碌碌的过着自己的大学生活。兼职，考研，各种社会所需证件，更有甚者忙学生工作，职位竞争。用一种近乎麻木的慌乱，给自己一种错觉：我们过得很充实。其实，当这样的青春走过后才发现：当初为了梦想而奋斗，最后奋斗的失去了梦想，那么何不静下心来。看看自己适合什么？给自己的生命定位，估值……
